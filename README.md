# Diwali_Sales_Analysis
Exploratory Data Analysis (EDA) of Diwali Sales Dataset using Python.

##  Project Overview
This project analyzes Diwali sales data using Python to identify customer purchasing behavior and business trends.

##  Objectives
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Customer Segmentation
- Sales Analysis
- Data Visualization

##  Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Female customers purchased more than male customers.
- Age group 26–35 contributed the highest sales.
- Uttar Pradesh, Maharashtra, and Karnataka generated the highest revenue.
- Food, Clothing, and Electronics were the top-selling categories.
